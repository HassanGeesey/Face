# Quotly

Quotation generation library for React Native.

## Features
- Modular data structure for quotations.
- Automatic calculation of totals.
- API integration with pdfgen.app.
- Offline-ready validation logic.

## Installation
```bash
npm install
```

## Usage
Refer to `example/QuotationView.js` for React Native usage.

## Development
Run tests:
```bash
npm test
```
